# ✧ Projeto: Operações Matemáticas, Hipotenusa e IMC ✧

Este projeto é uma aplicação web simples desenvolvida com **HTML, CSS e JavaScript**, com o objetivo de praticar lógica de programação e manipulação de dados por meio de cálculos matemáticos.

A interface permite que o usuário interaja diretamente com a página, inserindo valores nos campos de entrada e realizando diferentes tipos de cálculos através de botões.

## ✧ Funcionalidades

### Operações Matemáticas Básicas

➕ Soma  
➖ Subtração  
✖️ Multiplicação  
➗ Divisão  

### Cálculo da Hipotenusa
> Utilizando o **Teorema de Pitágoras**, o sistema calcula automaticamente o valor da hipotenusa a partir dos dois catetos informados.

### Cálculo de IMC
> O usuário informa sua altura e peso, e o sistema calcula o **IMC (Índice de Massa Corporal)** com base na fórmula matemática correspondente.


## ✧ Objetivo do Projeto

Aplicar conceitos fundamentais de:

- Manipulação do DOM  
- Funções em JavaScript  
- Captura de valores com `input`  
- Eventos com `onclick`  
- Estruturação básica de layout com HTML e CSS
